# movies-explorer-api


# Бэкенд проекта Movies
Дипломная работа для Яндекс Практикум

Проект - API для аутентификации пользователей и сохранения фильмов в сервисе Movies.


## Запуск проекта

`npm run start` — запускает сервер  
`npm run dev` — запускает сервер с hot-reload
